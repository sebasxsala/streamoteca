# Streamoteca

**Streamoteca** is an AI-powered movie and TV series discovery platform that leverages advanced machine learning techniques to deliver highly personalized content recommendations.

## Overview

This application combines modern web technologies with AI-driven features to create an intelligent streaming content discovery experience. Users can explore thousands of movies and TV shows, build personal watchlists, and receive recommendations tailored to their viewing preferences.

## AI Implementation Details

The application uses **vector embeddings** as the foundation for its intelligent features:

1. **Content Embeddings**: Each movie and TV show is processed through Google's text-embedding-004 model, which analyzes titles, descriptions, and genres to create high-dimensional vector representations

2. **Search Embeddings**: User search queries are converted into embeddings and compared against content embeddings using cosine similarity (via PostgreSQL's pgvector extension)

3. **User Profile Embeddings**: The system aggregates a user's interaction history (weighted by action type) to generate a dynamic user profile embedding that represents their preferences

4. **Recommendation Engine**: Uses vector distance calculations to find content most similar to a user's profile, filtered by genre and popularity when needed

This approach enables the platform to understand semantic relationships between content and user preferences, delivering recommendations that go beyond simple genre or keyword matching.
